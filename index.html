My Work

Starbucks GIS Cyprus

An application which displays the Starbucks Stores of Cyprus on GMap Components in a C# Windows Forms Application through GeoJson Datasets and collections stored in a MongoDB Database. CRUD operations are performed and the Republic of Cyprus polygon restricts the user from placing a store or a truck outside of it using the Ray Casting Algorithm.

Smart Hotel Monitoring System utilising Arduino R3 Microcontroller / C++

This project simulates a hotel monitoring system by retrieving real values from the Arduino Microcontroller and acting accordingly. This project incrorporates various Design Patterns and Secure Software Engineering Principles.

GP6 Medical Bot

A web application built in Anvil.works which dynamically populates the User Interface with Symptoms in the form of checkboxes, of which their Severity can be set. The user can select multiple symptoms and receive a recommendation using a predictive algorithm.

Flight Simulator Controller in C#.Net using Arduino Microcontroller.

This application acts as a client and connects to the server application which simulates the flight. The application is responsible for controlling the plane and numerous principles have been applied including Delegates, Events and Arduino Microcontroller Integration. The application also connects to a database in order to provide different airports and provide a flight time, simulating a flight experience, including take off and landing using various GUI techniques.
